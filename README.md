WordPress Plugins Scraper

Простой и эффективный веб-парсер для сбора данных о популярных плагинах с официального репозитория WordPress.org.

Note: Этот парсер является примером моего стиля написания кода, подхода к структурированию скриптов и обработки внешних данных.

🛠 Технологии
Python 3.x

BeautifulSoup4 — парсинг HTML контента.

Requests — выполнение HTTP-запросов.

CSV — хранение собранных данных.

🚀 Функционал
Имитация реального пользователя через User-Agent для обхода базовых защит.

Извлечение названий плагинов, их прямых ссылок и количества оценок.

Очистка и нормализация данных (удаление лишних символов, приведение рейтинга к числовому формату).

Защита от сбоев: обработка исключений при сетевых ошибках или изменениях в структуре страницы.

Автоматическое создание CSV-отчета с заголовками.
